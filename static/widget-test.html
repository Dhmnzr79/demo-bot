Тестовая страница виджета

Запусти демо-бота: python app.py (в этой демо-папке порт задан в .env: 9001). Открой эту страницу с того же хоста, например http://127.0.0.1:9001/static/widget-test.html — тогда POST /ask пойдёт на тот же origin (поле apiBase в конфиге пустое).

Разработка: сброс сессии — кнопка DEV · сброс sid слева сверху или Ctrl+Alt+R.

Если client_id не default, поправь JSON в #clinic-widget-config под свой ALLOWED_CLIENTS.

Для наглядного сырого JSON и кандидатов retrieval (как в старом тестере) открой отдельную dev-страницу: /static/debug/ask-inspector.html — её можно целиком удалить вместе с папкой static/debug/, на виджет она не влияет.
